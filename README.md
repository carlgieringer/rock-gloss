# RockGloss

This repo contains an Expo app and scraper supporting a rock climbing glossary.

## Scraper

Outputs updated terms to `rock-gloss-scraper/out/terms.json`
```
cd rock-gloss-scraper
npm install
npm run scrape
```

## Mobile app

```
cd rock-gloss-expo
yarn start
```
